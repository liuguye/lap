# Release Notes - Lap v0.2.3

Lap v0.2.3 expands Linux ARM availability, upgrades the image editing workflow, adds automatic folder scanning, and improves album performance and grid controls.

## 🚀 Highlights

- **Linux ARM Builds** — Added Linux ARM release builds, so Debian-based Linux users can download an `_arm64.deb` package alongside the existing `_amd64.deb` package.
- **Standalone Image Editor** — Edit Image now opens in a dedicated window with significant UI improvements, better window controls, preset polish, and safer save behavior.
- **Richer Histograms** — Histograms now support Luma, Red, Green, and Blue channels, including a hover legend and availability from the info panel.
- **Automatic Folder Scanning** — Folder sync now detects new, modified, deleted, and renamed files automatically, with non-blocking refresh behavior for album views. (#122)
- **Album Folder Controls** — Album views can include files from subfolders, individual subfolders can be excluded from queries, and large folder views load more efficiently without freezing. (#17, #96, #114, #125)
- **Grid Date Grouping** — When sorting by taken, created, or modified date, grid view can group files by day or month. Calendar view can also choose the date type used for sorting. (#116, #132)
- **Video Browsing Improvements** — Grid thumbnails can preview videos on hover, video thumbnails can cycle by refreshing file info, and video duration is available as a sort option. (#76, #102, #115)

## 🐞 Bug Fixes

- Improved load performance for album folder views containing a large number of files, preventing app freezes. (#114, #125)
- Fixed thumbnail generation failure for Nikon Z50 II `.NEF` RAW files. (#136)
- Fixed 1:1 actual-size zoom ignoring system DPI scaling. (#140)
- Fixed drag-and-drop errors on Linux.
- Improved multi-select behavior and dialog feedback. (#109)
- Fixed dedup panel state bleeding on folder switch and reduced UI blocking. (#119)
- Added a model download connection timeout and reduced the update check timeout. (#125)
- Fixed item reordering after file-import drag-and-drop handlers were added.
- Fixed swapped width and height dimensions in thumbnails for some portrait RAW formats.
- Fixed HEIC edit orientation so portrait HEIC files save correctly when exported as JPEG.
- Fixed a Linux build error. (#127)

## 🔧 Other Improvements

- **Permanent Delete** — Added a permanent delete option to bypass the system trash. (#113)
- **File Export** — Added file export with a native folder picker. (#105)
- **Tag Search** — Added a tag search filter to the Custom Tags tab. (#106)
- **Empty Library Welcome** — Added a welcome state for empty libraries.
- **Masonry Grid Option** — Merged the previous justify mode into grid style as a masonry view option.
- Improved empty states for filters and search results, search loading feedback, and content refresh race handling.
